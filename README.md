# 德州扑克命令行游戏

一个基于 TypeScript 开发的命令行德州扑克游戏，支持在终端、PowerShell 或浏览器控制台环境中运行。

## 功能特性

- **完整的德州扑克规则**：支持翻牌前、翻牌圈、转牌圈、河牌圈和摊牌阶段
- **ASCII 卡牌可视化**：使用 Unicode 符号和 ASCII 艺术展示 52 张标准扑克牌
- **AI 对手**：基于手牌强度、底池赔率和位置因素的智能决策系统
- **中文界面**：完整的简体中文界面和交互提示
- **多人游戏**：支持 2-8 名玩家同时游戏
- **严格类型检查**：使用 TypeScript 严格模式确保代码质量

## 安装与运行

### 环境要求
- Node.js >= 16.0.0
- npm 或 yarn

### 安装依赖
```bash
cd holdem-cli
npm install
```

### 编译项目
```bash
npm run build
```

### 运行游戏
```bash
npm start
```

### 开发模式
```bash
npm run dev
```

## 游戏操作

### 初始设置
运行游戏后，根据提示输入：
1. 玩家数量（2-8人）
2. 你的座位位置（1-N）

### 游戏动作
在你的回合，使用数字键选择动作：

| 数字 | 动作 | 说明 |
|------|------|------|
| 1 | 弃牌 | 放弃当前手牌 |
| 2 | 过牌 | 不下注，轮到下一位 |
| 3 | 跟注 | 跟上下注金额 |
| 4 | 加注 | 增加下注金额 |
| 5 | 全押 | 押上所有筹码 |

### 游戏流程
1. **翻牌前**：发放两张底牌，进行第一轮下注
2. **翻牌圈**：发出三张公共牌，进行第二轮下注
3. **转牌圈**：发出第四张公共牌，进行第三轮下注
4. **河牌圈**：发出第五张公共牌，进行最后一轮下注
5. **摊牌**：剩余玩家展示手牌，比较大小分配底池

## AI 决策系统

AI 对手基于以下因素做出决策：

### 手牌强度计算
- 根据当前手牌等级（高牌到皇家同花顺）计算基础强度
- 结合踢脚牌价值进行微调
- 范围：0.0（最弱）到 1.0（最强）

### 底池赔率
- 计算需要跟注的金额与底池比例
- 用于判断跟注是否具有正期望值

### 侵略性因子
- 基于玩家位置（后位更具侵略性）
- 范围：0.5 到 1.5

### 决策逻辑
| 调整后强度 | 动作 |
|-----------|------|
| < 赔率×0.5 | 弃牌/过牌 |
| > 0.8 | 加注/全押 |
| > 0.6 且无人下注 | 主动加注 |
| 其他 | 跟注/过牌 |
