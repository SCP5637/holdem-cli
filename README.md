# 德州扑克命令行游戏

一个基于 TypeScript 开发的命令行德州扑克游戏，支持在终端、PowerShell 或浏览器控制台环境中运行。

## 安装与自定义运行

### 环境要求
- Node.js >= 16.0.0
- npm 或 yarn

### 安装依赖
```bash
cd holdem-cli
npm install
```
*(如果只是想玩，打开目录里的exe文件即可)*
*(打开到holdem-cli目录下，目录输入栏内东西全删了输入cmd并回车即可在当前目录下打开bash，当然如果你知道这是什么你也可以很轻松的在其他的终端里打开)*

### 编译项目
```bash
npm run build
```

### 运行游戏
```bash
npm start
```
*(如果你发现无法运行，请检查是否npm i成功，以及是否npm run build成功)*

### 开发模式
```bash
npm run dev
```

## 游戏操作

### 初始设置
运行游戏后，根据提示输入：
1. 玩家数量（2-8人）
2. 你的座位位置（1-N）

### 游戏动作
在你的回合，使用数字键选择动作：

| 数字 | 动作 | 说明 |
|------|------|------|
| 1 | 弃牌 | 放弃当前手牌 |
| 2 | 过牌 | 不下注，轮到下一位 |
| 3 | 跟注 | 跟上下注金额 |
| 4 | 加注 | 增加下注金额 |
| 5 | 全押 | 押上所有筹码 |

### 游戏流程
1. **翻牌前**：发放两张底牌，进行第一轮下注
2. **翻牌圈**：发出三张公共牌，进行第二轮下注
3. **转牌圈**：发出第四张公共牌，进行第三轮下注
4. **河牌圈**：发出第五张公共牌，进行最后一轮下注
5. **摊牌**：剩余玩家展示手牌，比较大小分配底池

## 功能特性

- **完整的德州扑克规则**：支持翻牌前、翻牌圈、转牌圈、河牌圈和摊牌阶段
- **ASCII 卡牌可视化**：使用 Unicode 符号和 ASCII 艺术展示 52 张标准扑克牌
- **AI 对手**：基于手牌强度、底池赔率和位置因素的智能决策系统
- **中文界面**：完整的简体中文界面和交互提示
- **多人游戏**：支持 2-8 名玩家同时游戏
- **严格类型检查**：使用 TypeScript 严格模式确保代码质量

## 项目结构

```
holdem-cli/
├── src/
│   ├── types/          # 类型定义
│   │   ├── card.ts     # 卡牌类型（花色、点数）
│   │   └── game.ts     # 游戏状态类型
│   ├── core/           # 核心游戏逻辑
│   │   ├── deck.ts     # 牌组管理（创建、洗牌、发牌）
│   │   ├── gameState.ts    # 游戏状态管理
│   │   ├── handEvaluator.ts # 手牌评估系统
│   │   └── aiPlayer.ts      # AI 对手逻辑
│   ├── ui/             # 用户界面
│   │   ├── cardRenderer.ts  # 卡牌渲染
│   │   ├── gameRenderer.ts  # 游戏界面渲染
│   │   └── inputHandler.ts  # 输入处理
│   └── index.ts        # 程序入口
├── package.json        # 项目配置
├── tsconfig.json       # TypeScript 配置
└── README.md           # 项目说明
```
